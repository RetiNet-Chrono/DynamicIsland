# Windows灵动岛（纯Win32 API实现）

简洁清晰的项目描述：一款基于纯Win32 API开发的Windows端灵动岛模拟应用，采用C++20标准编写，无任何第三方依赖库，轻量化还原移动端灵动岛交互与视觉效果，原生适配Windows系统特性。

## 特性
- 纯原生实现：基于Win32 API手工绘制UI、处理窗口交互，无MFC/Qt/SDL等第三方框架，极致轻量化
- C++20特性加持：充分利用C++20的新特性优化代码逻辑与内存管理
- 拟态交互体验：模拟移动端灵动岛核心交互，支持点击切换状态、拖动调整位置、圆角/透明背景渲染
- 低资源占用：无冗余依赖，运行时内存占用＜5MB，无后台常驻进程，关闭即完全释放资源
- 系统版本适配：兼容Windows 10 1903+ / Windows 11（推荐使用Windows 11）
- 自动注册开机自启动，无需手动配置

## 环境要求
- 编程语言版本：C++20（需编译器完整支持C++20标准）
- 依赖工具/框架：无第三方库，仅依赖Windows系统自带Win32 API（User32.dll、Gdi32.dll、D2d1.dll等）
- 编译环境：MSVC 2022+（Visual Studio 2022 带C++桌面开发组件）
- 系统要求：Windows 10 1903+ / Windows 11（x64架构）
